# Blinkit-Grocery-Sales-Analysis
SQL analysis of grocery retail data to identify sales trends, item visibility, and outlet performance using MySQL.

# 🛒 Blinkit Grocery Analysis

### Project Overview
This project performs an in-depth analysis of grocery data to understand sales patterns across different outlets and product categories.

### Key Features
* **Inventory Analysis:** Evaluated item weights, fat content, and visibility.
* **Outlet Performance:** Segmented sales by outlet size, location type, and establishment year.
* **Detailed Metrics:** 50+ SQL queries covering basic counts to complex group-wise aggregations.

### Tech Stack
* **Database:** MySQL
* **Key Skills:** Data Cleaning, Aggregations (SUM, AVG), and Grouping.
